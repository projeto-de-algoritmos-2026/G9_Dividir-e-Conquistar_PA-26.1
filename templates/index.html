      Contagem de Inversões O(n log n)

      Modificação do Merge Sort. Durante o merge de dois sub-arrays, sempre que um elemento do sub-array direito for menor que um elemento do esquerdo, todos os elementos restantes no esquerdo formam inversões com ele. 

      def merge_count(arr):
    mid = len(arr) // 2
    left, inv_L = merge_count(arr[:mid])
    right, inv_R = merge_count(arr[mid:])
    merged, inv_M = merge(left, right)   # conta cruzamentos
    return merged, inv_L + inv_R + inv_M

      No contexto do dashboard: cada inversão = um ônibus que chegou fora da ordem planejada.

      Mediana das Medianas O(n)

      Garante seleção do k-ésimo menor em tempo linear no pior caso (diferente do QuickSelect que é O(n²) no pior).

      1. Dividir em grupos de 5 elementos
      2. Encontrar mediana de cada grupo (sort trivial)
      3. Chamar recursivamente para achar a "mediana das medianas"
      4. Usar esse valor como pivô garantido
      5. Particionar e recursar no sub-array correto

      Aplicado aqui: encontra o atraso mediano ignorando outliers extremos.
